# ClientCortex

A sleek, modern tech landing page for ClientCortex - the intelligent client management platform.

## Features

- Modern, responsive design with light backgrounds and blue color scheme
- Fast-loading CSS animations (fade, slide, float, pulse effects)
- Intersection Observer for scroll-triggered animations
- Mobile-first responsive layout
- Sections: Hero, Features, How It Works, Benefits, Testimonials, Pricing, CTA, Footer

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript
- Google Fonts (Inter)
- No external dependencies for fast loading

## Getting Started

Simply open `index.html` in a web browser to view the landing page.
